# Project Overview  
This project is a Django-based web application that includes user registration, email verification, login and profile creation with interest selection for recommendations (for now)

Planned and implemented multi-module platform functionality for social interactions - including registration, chat, gallery, relations, and marketplace features. Created full use case diagrams and designed the project structure for backend modules.
